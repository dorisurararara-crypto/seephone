# 빡신 프로젝트 — 앱 3종 개요

> 1인 개발 + 서버 0원 + Flutter 크로스플랫폼 + 한국 시장 블루오션

## 공통 원칙

1. 서버 개발 없음 (운영비 0원)
2. iOS / Android 동시 출시 (Flutter 1코드베이스)
3. 한국 앱스토어 블루오션 타겟
4. 자발적 바이럴 (스크린샷·영상으로 공유될 자산을 앱이 만들어 줌)
5. 스마트폰 자체 하드웨어/센서 + 로컬 데이터만 활용

## 개발 순서

| 우선순위 | 앱 | 디렉토리 | 상태 |
|---|---|---|---|
| 1 | 빡신 (디지털 무당) | `bbaksin/` | 코드 골격 + 5 테마 + 멘트 + 캡처/공유 ✅ |
| 2 | 동공 지진 탐지기 | `pupil/` | 코드 골격 + 카메라 + LieDetector + 결과 화면 ✅ |
| 3 | 분노 발전소 | `anger/` | 코드 골격 + 가속도 측정 + AngerCalc + 결과 화면 ✅ |

**순서 근거**: 코드 가장 단순 → 가장 빠른 첫 출시 → 광고 SDK·앱스토어 운영 노하우 확보 후 다음 앱으로 재활용.

## 1. 빡신 (디지털 무당 / 팩폭 타로)

**가제**: 빡신
**컨셉 한 줄**: 고민 적고 폰 흔들면, 욕쟁이 무당이 부적으로 팩폭 날린다.
**서브카피**: 진지한 타로 NO. 매운맛 점집 OK.

### 핵심 기능
1. 고민 입력 (텍스트 한 줄)
2. 폰을 위아래로 3번 흔들기 = 점치는 행위
3. 굿판 애니메이션 후 부적 등장
4. 부적 = 키치한 한국 전통 부적 배경 + 팩폭 멘트 합성
5. 부적 갤러리 저장 / SNS 공유

### 신박한 포인트
- 진지한 타로/사주 앱 NO. 건방진 'MZ 감성 디지털 점집'
- 부적 자체가 공유 자산 (인스타 스토리 / 카톡 프사 / 단톡방)

### 서버비 0원 이유
- 팩폭 멘트 1,000개 = 앱 번들 내 JSON
- 부적 배경 이미지 ~20장 번들 (이미지 + 런타임 텍스트 합성으로 사이즈 최소화)
- 모든 점치기 = 로컬 랜덤

### 수익 모델
- 무료 부적 하루 N개, 추가 부적은 30초 전면 광고 시청 후 해금
- AdMob 전면 광고 + 보상형 광고

### 바이럴 포인트
- 부적 하단에 앱 이름 (빡신) 워터마크 필수
- 갤러리 저장 → 스토리/프사 → 친구 호기심 → 다운로드 루프

### 비주얼 가이드 요약
- 컬러: 부적 빨강 `#C8102E`, 한지 베이지 `#F4E4BC`, 먹 검정 `#1A1A1A`, 부적 노랑 `#F4D35E`
- 폰트: 정묵바위체 / 양진체 / 배달의민족 도현체 (붓글씨 톤)
- 부적 배경: 9:16 세로 비율, 중앙 60%는 텍스트용으로 비워둠

## 2. 동공 지진 탐지기 (얼굴 인식 진실게임)

**컨셉**: 카메라로 친구 눈동자를 분석하는 B급 거짓말 탐지기.

### 핵심 기능
1. 전면 카메라로 얼굴 트래킹 (인스타 필터처럼 박스 표시)
2. 질문 던진 후 친구가 답하는 동안 3초 스캔
3. 눈동자 움직임 + 안면 미세 흔들림 분석
4. 결과: "동공 지진 진도 8.0! 새빨간 거짓말!"

### 기술 (서버비 0원)
- ML Kit (Android) / Vision (iOS) — 100% 로컬 얼굴 트래킹
- Flutter: `google_mlkit_face_detection` 플러그인

### 핵심 설계 룰
- **결정론적 결과**: 같은 얼굴 + 같은 질문 = 같은 결과 (재현성 = 신뢰감)
- 실제 신호 (눈 깜빡임 빈도, 머리 미세 회전, 웃음 detection) 3~4개를 가중치로 섞어서 판정 생성
- 앱스토어 등록: "엔터테인먼트" 카테고리 명시 (Apple Guideline 4.3 / 2.3 회피)

### 바이럴 포인트
- 친구 얼굴 + 거짓말 판정 = 화면 녹화 → 틱톡/릴스/숏츠

### 수익 모델
- 결과 보기 전 전면 광고
- "정밀 분석 모드" 보상형 광고

## 3. 분노 발전소 (스마트폰 지진계 & 타격기)

**컨셉**: 빡침을 물리적 에너지로 측정하는 스트레스 해소 앱.

### 핵심 기능
1. [10초 분노 방출] 버튼
2. 폰을 흔들거나 화면을 연타
3. 가속도 센서 + 터치 이벤트로 분노 에너지 수치화
4. 결과: "당신의 분노는 1,500W. 전자레인지 3분 작동 가능"
5. 킹받는 멘트: "고작 10W? 모기 날갯짓 수준의 분노네요"

### 기술 (서버비 0원)
- `sensors_plus` (가속도 센서)
- 터치 이벤트 카운트
- 100% 로컬

### 안전 가이드
- 온보딩에 "폰을 꽉 잡으세요" 면책 문구
- 손목 스트랩 사용 권장 안내

### 수익 모델
- 분노 측정 결과 보기 전 전면 광고
- "더 강한 샌드백 이펙트" 해금 = 보상형 광고
- (선택) 로컬 랭킹 보드 — SharedPreferences로 서버 없이 구현
