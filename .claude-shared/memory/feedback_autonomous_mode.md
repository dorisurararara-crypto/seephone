3개 앱 (빡신, 동공지진, 분노발전소) 모두 완성될 때까지 **사용자에게 묻지 말 것**. Mac과 Windows Claude 둘이 판단해서 진행.

**Why:** 사용자가 2026-04-28 23:30경 "앞으로는 나한테 묻지말고 3번째앱까지 너네 둘이 판단하에 완성시켜놔" 라고 명시. 묻기 위해 멈추는 게 가장 큰 비용.

**How to apply:**
1. 막히면 먼저 로컬 머신을 직접 탐색 (`find`, `ls`, `which`, `ps`, `~`/`Library`/`Applications` 등) — 파일·도구·자격증명이 어디 있는지 직접 찾기
2. 그래도 안 풀리면 WebSearch / 커뮤니티 검색으로 해결법 찾기
3. **진짜로** 사용자만 할 수 있는 일 (Apple ID 2FA, 본인 인증, 비밀번호 직접 입력, Apple Developer 신규 가입 등) 인지 검증
4. 진짜 사용자 액션 필요한 항목은 → "사용자 대기" 목록에 기록 후 **그 외 모든 가능한 작업을 다 진행**한 다음에만 보고
5. Windows Claude한테도 같은 mandate 적용 (HANDOFF.md 통해)

**예외 (사용자에게 보고는 해도 됨):**
- 출시 직전 최종 점검 (앱스토어 메타데이터 확인 등)
- 결제 흐름처럼 1-2단계 본인 확인이 정말 필요한 작업
- 큰 디렉션 변경 (앱 컨셉 자체를 바꾸는 등) — 이건 묻기 전에 하지 말 것
