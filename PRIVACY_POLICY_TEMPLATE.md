# 빡신·동공지진탐지기·분노발전소 개인정보 처리방침 (템플릿)

> 3개 앱 모두 동일한 정책. 출시 시 각 앱 페이지에 공개 (App Store Connect 등록 필수 항목).
> 호스팅 권장: GitHub Pages 또는 Notion public page.

## 본 앱은 개인정보를 수집하지 않습니다

본 앱은 사용자의 입력·측정·이용 기록을 **모두 단말기 내부에만 저장**하며, 외부 서버로 전송하지 않습니다.

## 처리 항목

| 항목 | 처리 방식 | 외부 전송 |
|---|---|---|
| 사용자가 입력한 고민·질문 텍스트 | 단말기 메모리 내 일시 보관, 앱 종료 시 폐기 | ❌ 안 함 |
| 측정 데이터 (가속도/터치 빈도/얼굴 랜드마크) | 단말기 내 처리 후 즉시 폐기 | ❌ 안 함 |
| 부적/결과 이미지 | 사용자 명시적 저장 시에만 사용자 갤러리에 저장 | ❌ 안 함 |
| 설정 (선택 테마, Pro 구독 상태) | 단말기 내 SharedPreferences 에 저장 | ❌ 안 함 |

## 권한 사용

- **카메라** (`동공 지진 탐지기`만): 친구 얼굴 분석을 위해 전면 카메라 사용. 영상은 **저장하지 않으며 외부 전송도 안 함**. 분석은 단말기 내 ML Kit (Google) 가 100% 로컬로 수행.
- **사진 라이브러리 추가**: 결과 이미지를 사용자가 명시적으로 "저장" 누를 때만 사진첩에 추가.
- **진동**: 흔들기 / 결과 도착 시 햅틱 피드백.
- **가속도 센서** (`분노 발전소`만): 분노 측정. 데이터는 단말기 내 처리 후 즉시 폐기.

## 광고

본 앱은 **Google AdMob** 광고를 표시합니다. AdMob 은 자체 정책에 따라 일부 정보를 처리할 수 있습니다:
- AdMob 정책: https://policies.google.com/technologies/ads
- 사용자는 iOS 추적 권한 (App Tracking Transparency) 으로 광고 ID 사용 거부 가능.
