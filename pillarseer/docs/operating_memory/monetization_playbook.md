## 핵심 원칙 — "합당함"

1. **무료도 하나의 완성된 사주 앱.** 무료 사용자가 앱을 계속 쓸 진짜 이유가 있어야 한다.
2. **프리미엄은 "막힌 앱 정상화"가 아니라 "더 깊이 여는 확장팩".** 유료를 쓰면 확실히 더 좋다는 체감.
3. **본문 중간 truncate/blur 금지.** 카테고리·항목 단위로 잠근다. ("여기부터 유료"는 미끼로 느껴짐.)
4. 잠긴 항목은 제목 + 짧은 설명 + 프리미엄 CTA 만 표시. 기능 고장/미완성처럼 보이면 안 됨.
5. **"준비 중" 기능을 유료 혜택으로 팔지 않는다.** 특히 최애의 사주 준비 중 193명은 paywall 혜택·문구에서 제외.

### ① 내 사주 17카테고리 — 5 무료 / 12 프리미엄

**무료 5개** (= "나는 어떤 사람인가"의 완결):
핵심 성향 · 오행 균형 · 십신 성향 · 강점/주의점 · 오늘 바로 써먹는 조언

**프리미엄 12개** (= "내 인생 영역들"):
연애 · 결혼 · 가족/자녀 · 일/커리어 · 재물 · 학업/성장 · 인간관계 ·
컨디션/생활 리듬 · 대운(10년 흐름) · 올해 흐름 · 월별 흐름 · 깊은 종합 결론

주의: 건강은 "건강 진단"이 아니라 "컨디션/생활 리듬" 톤. 무료 카테고리 본문이 유료보다
짧아도 되지만 **중간 절단 금지**.

## 결제의 영구성 — 기기 변경 대응

- non-consumable IAP 는 **사용자 Apple ID 에 영구 귀속** (기기 아님).
- 폰 교체·재설치·새 기기 → 같은 Apple ID 면 무료로 다시 받음.
- 로컬 저장 entitlement 는 **캐시**일 뿐, 진짜 source of truth 는 App Store.
- **앱 시작 시 조용히 자동 복원** (StoreKit current entitlements 조회) — 사용자가 버튼 누를
  필요 없이 자동 unlock. 자동 복원이 못 잡는 예외만 수동 "구매 복원" 버튼이 받음.
- 수동 "구매 복원" 진입점 **2곳**: paywall 하단 + 설정 화면. (Apple 심사가 non-consumable
  에 restore 를 의무 요구 — 없으면 거절.)
- 사주 입력 데이터(생년월일 등)는 로컬 저장 — 기기 변경 시 iCloud 백업으로 따라오거나,
  재입력(결정론적이라 결과 동일). 손실 아님.

## Paywall UX

- 진입: 무료 사주 정독 → 잠긴 카테고리 묶음 노출 → "더 깊게 보기" → paywall.
- 구매 안 해도 무료 앱으로 복귀 가능 ("지금은 무료로 계속 보기").
- 업셀 지점: 내 사주 무료 리포트 끝 / 잠긴 카테고리 탭 / 신년 4월 이후 / 궁합 2·3번째
  관계 / 전생 추가 편 / 자미두수 상세 / 설정의 프리미엄팩.
- **금지 지점**: 앱 첫 실행 직후 전면 paywall / 생년월일 입력 후 결과 직전 / 무료 결과
  첫 문단 중간 차단 / "준비 중" 클릭 시 결제 유도.

### 문구 (확정)
- 헤드라인: `내 사주를 더 깊게 열어보세요`
- 서브: `무료 리포트는 계속 사용할 수 있어요. 프리미엄팩은 긴 해석과 추가 리포트를 한 번에 여는 단건 구매입니다.`
- CTA: `프리미엄팩 열기`
- 보조: `지금은 무료로 계속 보기`
- 가격 보조: `한 번 구매하면 계속 사용할 수 있어요.` (가격은 StoreKit localized price)
- restore: `이미 구매하셨나요? 구매 복원`
- 성공 toast: `프리미엄팩이 열렸습니다.`
- 취소 toast: `구매가 취소되었습니다.`
- restore 실패: `복원할 구매 내역을 찾지 못했어요.`
