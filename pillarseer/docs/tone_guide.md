## 0. 페르소나
- 타겟: 한국 MZ 중학생 K-POP 팬 (Co-Star 글로벌 진출 전 한국 검증 단계)
- 결정 빠르고 헷지 표현 싫어함. **단정·구체·소름** 콘텐츠 선호.

## 1. 어조의 정의

**확정 단정조 = "당신은 ~다 / 오늘 ~한다 / 곧 ~온다" 평서·예측 단정.**

콜드리딩이 통하는 이유는 high-base-rate(많은 사람에게 적용되는) 진리를 "마치 너만 알고 있다"는 듯 단정하기 때문. "할 수 있어요" 는 빠져나갈 길을 열고, "한다" 는 닫는다.

## 2. 금지 표현 (자동 검출 — `transform_tone.py` enforce)

| 카테고리 | 금지 어휘 | 페널티 |
|---|---|---|
| 헷지 종결 | `있어요 / 없어요 / 할 수 있어요 / 일 수 있어요 / 것 같아요 / 것 같다 / 일지도 / 편이에요 / 편이다 / 봅니다` | -0.5/건 |
| Advisory | `도움됩니다 / 추천드려요 / 좋습니다 / ~ 게 좋아요 / ~ 게 좋다 / ~ 편이 좋다` | -0.5/건 |
| 보호자체 | `조심하세요 / 지키세요 / 주의하세요 / 신경 쓰세요 / 챙기세요 / 아끼세요` | -0.5/건 |
| 점쟁이체 | `본질 / 결 (한자 jargon 의미) / 정수 / 천명 / 운명적으로 / 사주의 결 / 운기 / 기운이 (advisory 맥락)` | -0.5/건 |
| K-pop 추상 비유 (kpop_compat 외부) | `무대 / 퍼포먼스 / 팬덤 / 응원 / 센터 / 컴백 / fancam / 카메라 워크 / 셋리스트 / 무대 위` | -1.0/건 |

## 3. 권장 표현 (단정 + 콜드리딩 + 시점)

| 패턴 | 예시 |
|---|---|
| 단정 평서 | "당신은 ~다", "너는 ~ 사람이다", "오늘 너는 ~한다" |
| 시점 단정 예측 | "이번 주 ~ 일이 온다", "곧 누가 ~ 물어본다", "이번 달 안에 ~", "40대 전후로 ~", "다음 ~ 에" |
| 콜드리딩 hit (high-base-rate 단정) | "지난주에 친구랑 한 번 연락 끊겼다 다시 됐다", "최근 자기 전 핸드폰 오래 본다", "한 명 신경 쓰이는 사람 떠오른다", "엄마/아빠 말이 아직 머릿속에 한 줄 남았다", "충동적으로 결제한 게 있다" |
| 직설 친근 (반말 톤 살짝) | "네 안에 두 개 있어. 하나는 ~ 하나는 ~", "그거 알아? ~" |

## 4. 분량·호흡 (entry 단위)

- 한 entry = **7~12 문장**, 각 문장 **30~80자**
- 첫 문장 = **단정 평서 hook** ("당신은 ~다")
- 2~3 문장째 = **콜드리딩 hit 1개** (high-base-rate 단정)
- 4~5 문장째 = **시점 단정 예측 1개** ("이번 달 ~", "곧 ~")
- 마지막 문장 = **단정 결론** (advice 형태 금지)

## 5. Before/After 예시 5쌍

### 예시 1 — 甲申 (Wood Monkey) dayMasterDeep

**Before**: "당신은 강함과 부드러움이 같이 있는 사람이에요. 밖에선 단단해 보이지만 안에선 의외로 다정한 면이 있고, 결정은 빠르게 내리는 편이에요. … 다만 계산이 과하면 신뢰가 늦게 열려요."

**위반**: "있는 사람이에요(헷지)", "다정한 면이 있고(헷지)", "내리는 편이에요(헷지)", "신뢰가 늦게 열려요(완곡)" — 헷지 4건, 콜드리딩 hit 0건.

**After**: "당신은 안과 밖이 다르다. 밖에선 단단한 사람으로 보이지만, 안에서는 의외로 무른 곳이 있다. 결정은 빠르다 — 그런데 그 결정을 혼자 두세 번 되감는다. 지난주에도 한 번 그랬지. 가까운 사람 한 명한테 '괜찮아?' 묻고 싶었는데 안 물었다. 너는 계산이 많은 사람이고, 그게 너의 무기다. 하지만 그 계산을 상대가 눈치챈 순간 신뢰는 닫힌다. 이번 달 안에 한 번, 평소 너답지 않게 먼저 연락하는 일이 생긴다. 그게 정답이다."

**개선** (tool 검출 기준):
- 단정 종결 8/8 문장 (다·한다·닫힌다·정답이다·그랬지 등 — `ASSERTION_VERB_ENDINGS` 확장)
- 콜드리딩 hit (anchor + 구체 명사 + 단정 동시 충족):
  - "지난주에도 한 번 그랬지" → anchor `지난주` + 구체 `한 번` + 단정 종결 `그랬지` (3 조건 모두 충족)
  - "이번 달 안에 한 번, 평소 너답지 않게 먼저 연락하는 일이 생긴다" → anchor `이번 달` + 구체 `한 번 / 먼저 연락` + 단정 종결 `생긴다`
- 시점 예측: "이번 달 안에" (`이번 달`)
- 9 문장 (7~12 범위)
