# V1 — 격국·십신·용신 결합 본문

## 원천
- unsin.co.kr 자체 페이지 (홈) — 격국·용신 한자 jargon 전혀 노출 X. "당신의 결혼상대 외모/직업" 식 결과 중심 제목으로 sublimate.
- 위키 "용신": 격국용신 (격국 기반 용신 선택) = 정격 8개에 따라 용신 선호 다름.
- 도화로운/사주플러스 칼럼: 격국 + 십신 결합 본문이 같은 정관/식신 이라도 "격국 위배 / 격국 따라감" 따라 의미 정반대.

## 왜 잘 본다고 느끼게 하는가
1. 격국 (정관격/식신격/정인격 등) 을 사용자에게 직접 노출 X → 한자 jargon 피로 0
2. "당신은 책임감으로 빛나요 + 단, 식신이 견제하면 자기 표현이 살아요" 식 격국 + 십신 결합 본문으로 풀이
3. 같은 정관 일진이어도 "정관격 + 식신 충" vs "정관격 + 정관 강화" → 본문 표현 정반대 (운세의신 후기 후한 평가 원인 1)

## 우리 앱 도입 plan (sprint 4 예정)
- 신규 pool `gyeokguk_join_pool.json`: 격국 8 × 십신 그룹 (관성/식상/재성/인성/비겁 5) × 용신 5 = ~200 key
- key 형식: `{격국short}|{십신그룹}|{용신}` — 예: `정관격|관성|木`
- 본문 ko/en — 한자 jargon 0 / MZ 친밀 해요체
- partial match priority: 정확 → 격국+십신 → 격국만 → 정적 fallback
