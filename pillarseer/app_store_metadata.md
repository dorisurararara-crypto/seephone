## Description (한국어 — codex 권장 카피)

```text
Pillar Seer는 사주를 처음 보는 사람도 쉽게 이해할 수 있는 개인 운세 앱입니다.

생년월일시를 바탕으로 나의 성향, 인간관계, 오늘의 흐름, 연애운, 중요한 시기를 부드럽고 직관적인 말로 풀어드려요.

어려운 한자와 전문 용어 대신,
"오늘은 어떤 말을 조심하면 좋을지"
"연애에서는 어떤 패턴이 반복되는지"
"올해 어떤 변화가 다가오는지"
바로 이해할 수 있게 보여줍니다.

주요 기능:
- 나의 본성, 관계 성향, 오행 균형 풀이
- 오늘의 운세와 시간대별 흐름
- 연애운, 궁합, 중요한 날짜 리포트
- 10년 인생 흐름과 올해의 분위기
- 사주 초보를 위한 쉬운 설명

매일 아침, 오늘 조심할 것만 가볍게 확인해보세요.
```

Every morning at 8 — a single line on what to watch out for. That's it.

## App Store Connect API 작업 순서 (TestFlight 외부 베타 → 정식 출시)

### TestFlight 외부 베타 review (Apple 정책: 1 train = 1 build review 동시)
1. Build #3 review 통과 대기 (자연 큐)
2. OR Build #3 cancel → submit Build #6 (사용자 mandate "1등이야 할 때까지 + 출시" 라면 #6가 1등 quality 이므로 우선)
3. `ruby scripts/submit_external_beta.rb 6` 한 줄로 submit

### App Store 정식 출시 (별도 단계)
1. ASC 콘솔에서 "+ Version" → 1.0.0 (Build #6 선택)
2. 위 description / keywords / screenshots 입력
3. Age Rating questionnaire
4. App Review Information 입력 (테스트 계정 불필요 — 로그인 X)
5. Submit for Review
6. 24-48h 후 게시 ("Manually release this version" 선택 시 사용자가 게시일 결정)
