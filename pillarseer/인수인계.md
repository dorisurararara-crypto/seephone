## 6. 알려진 트랩

### 6.1 Xcode 26 altool silent fail
같은 팀 비슷한 prefix bundle ID 다수 시 altool 이 잘못된 app 에 attach. **`--apple-id <numerical>` 명시 필수**. `deploy_testflight.sh` 에 이미 적용.

### 6.2 ASC `betaBuildLocalizations` whatsNew 길이
1009~1205자 PATCH → HTTP 409. **~500자 안에서 200 OK**. Release note 작성 시 길이 한도 둘 것.

### 6.3 flutter build ipa export 단계 fail 정상
`flutter build ipa --release` 의 export 단계는 cert 부족으로 fail (정상). `deploy_testflight.sh` 가 `|| true` 처리 후 `xcodebuild -exportArchive -authenticationKey*` 로 ASC API key 즉석 발급 — 실제 IPA 그때 생성.

### 6.4 진태양시 보정
서울 longitude 127.5° vs 일본 동경 135° = 약 32분 차. 사주 정통은 진태양시 채택. `ManseryeokService.seoulLongitudeOffsetMinutes = -32`. 절기 경계 비교 시 진태양시 보정된 datetime 사용.

### 6.5 시뮬 PNG Read 한도
한 턴에 1-3장. 큰 PNG 는 `sips -Z 600 in.png --out small.png` 후 Read. 텍스트 로그 (`xcrun simctl spawn booted log stream`) 우선.

### 6.6 codex 호출 hang
`codex exec "$(cat ...)"` X — 큰 prompt 시 hang. **stdin pipe** 필수.

### 6.7 spaceauth cookie ~30일 유효
만료 시 `fastlane spaceauth -u zkxmel@naver.com` 1회 실행 안내. TestFlight 자동화 mandate 도달한 경우만.
