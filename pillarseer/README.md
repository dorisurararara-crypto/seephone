## 🚀 빠른 요약

### 무엇을 만드는가
- **앱 이름 확정**: **Pillar Seer** (Four Pillars of Destiny + Seer = "운명의 4기둥을 읽는 예언자")
- **도메인**: `pillarseer.com` / `pillarseer.app`
- **소셜**: `@pillarseer` (Insta/TikTok/YouTube)
- **Bundle ID**: `com.ganziman.pillarseer`
- **핵심 화면 5개**: 온보딩 / 일생 사주 / 데일리 운세 / 단건 결제 콘텐츠 / 마이페이지
- **콘텐츠**: 일생 사주, 데일리 운세, 토정비결, 궁합, 택일, 꿈해몽

### 어떻게 돈을 버는가
- **무료 후크**: 일생 사주 기본 (성격/기질) 무료 공개
- **월 구독 ($4.99/월, $39.99/년)**: 데일리 상세 운세 + 10년 대운 + 광고 제거
- **단건 결제 ($9.99~$14.99)**: 토정비결, 심층 궁합, 비즈니스 택일
- **광고**: 무료 사용자 한정 (배너 + 보상형)

### 누구에게 파는가
- **타겟 1**: K-pop 팬 (대학생, 미국 — Chloe)
- **타겟 2**: 사주/타로 관심 직장인 (30대, 유럽 — Elena)
- **타겟 3**: K-drama 팬 (40대, 동남아 — Mei)

### 어떻게 알리는가
- **K-pop 아이돌 사주 클립** (BTS V "Born to be a star Saju" 등 50개)
- **K-drama 무속 장면 분석** (파묘, 환혼, 악귀)
- **TikTok/Insta 자가 운영** (매일 1~2개 숏폼)
- **D-day 인플루언서 10명 동시 시딩**

### 언제 출시하는가
- **Week 1**: MVP (온보딩 + 일생 사주 + 데일리 1화면 + Analytics)
- **Week 2~4**: 풀 (Auth + 구독 + 단건 결제 + 푸시)
- **Month 2~3**: 글로벌 (다국어, 지역 가격, K-pop 랜딩 플로우)
